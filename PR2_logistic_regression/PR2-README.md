# Лабораторная работа №2

В каталоге реализуется:

- логистическая регрессия для `diabetes.txt`;
- обучение методом градиентного спуска без `sklearn`;
- разбиение на обучающую и тестовую выборки;
- оценка точности классификации;
- отбор признаков по критерию CFS;
- сравнение с наивным отбором признаков;
- построение тепловой карты корреляций.

## Файлы

- `Practice2_logistic_regression.py` — основной скрипт;
- `data/diabetes.txt` — входной набор данных;
- `results/` — папка с результатами после запуска.

## Запуск

```powershell
python .\PR2_logistic_regression\Practice2_logistic_regression.py
```

## Что сохраняется

После запуска скрипт:

- выводит итоговые метрики и выбранные признаки в консоль;
- сохраняет `results/correlation_heatmap.png` — тепловую карту корреляций.
